# Engineering Portfolio Website This is a static portfolio site like the screenshots you shared. It uses only: - HTML - CSS - JavaScript That means it can be hosted for free with GitHub Pages. ## Edit Your Info Open `index.html` and change: - `Your Name` - `you@example.com` - LinkedIn links - Berkeley, CA - The short bio Open `script.js` and edit the `projects` array to change: - Project names - Years - Categories - Descriptions - Tags ## Add Your Real Images Right now the cards use generated CSS artwork so the design looks complete. To use real images, add an `images` folder and replace the `.project-image` background logic with normal `` tags. ## Put It On GitHub Pages 1. Create a new GitHub repository named `portfolio`. 2. Upload `index.html`, `styles.css`, `script.js`, and this README. 3. Go to repository `Settings`. 4. Click `Pages`. 5. Under `Build and deployment`, choose `Deploy from a branch`. 6. Select the `main` branch and `/root`. 7. Your site will appear at: ```text https://yourusername.github.io/portfolio/ ```
